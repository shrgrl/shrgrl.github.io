READING PART

There are many types of questions in the reading section:

  1. Factual Information Questions : This is about in the paragraph some real information. Probably, this question will start with “Acording to paragraph…”. You should read the paragraph and there will be a significant information in a sentence. You should do notice and find this answer.
  2. Negative Factual Information Questions : This question type is similar with first one. As you can see from the title you should find the wrong option. The paragraph has some information and this question ask you wrong one. Be careful and find it. And don’t forget reading question. Because of I didn’t read thoroughly and tried find correct opinion.Therefor, I did a lot of mistake. Probably this question sentence will involve like “NOT, EXCEPT…” key words.
  3. Inference Questions : In kind of this question, author tells us some information. But the author doesn't give detail. We need to find the desired answer from the implicit information given by the author.
  4. Rhetorical Purpose Questions : This is a similar type to the previous one. This question asks you to make an inference based on the implicit information provided by the author. Generally this question starts with “WHY…?”.
  5. Vocabulary Questions : In this question, there is a short pasage and some vocabulary. This question asks us for the vocabulary that corresponds to the closest meaning of the vocabulary in the sentence.
  6. Reference Questions : This question asks us reference something or someone. The paragraph says us some topic and someone or something in this topic and last it asks us “The word “ X ” in the passage refers to . . .”.
  7. Sentence Simplification Questions : In this question, there is a sentence in passage excerpt a sentence highlited. And question asks us, which sentence is same meaning with highlited sentence.
  8. Insert Text Questions : In this type of question, you are given a new sentence and are asked where in the passage it would best fit.
  9. Prose Summary Questions : In this type of question, you should choose summary ideas in the passage.
  10. Fill in a Table Questions : In this kind of question, you will be given a partially completed table based on information in the passage. Your job is to complete the table by clicking on cor-rect answer choices and dragging them to their correct locations in the table.
