Creating MySQL Database Connection with PHP

In projects we write using PHP, database management operations are usually performed through MySQL. There are two methods for connecting to an SQL database with PHP: MySQLi and PDO. First, we need to create a MySQL database. We had previously provided information about creating a database in another project. You can access it here. After creating the database, we can start writing PHP code.

Database connection is provided by creating a variable object:

{% highlight javascript %} $my_Object = new Object(); {% endhighlight %}

Let's examine an example PHP code:

{% highlight javascript %} $conn = mysqli_connect("localhost", "my_user", "my_password", "my_database"); if (!$conn) { die("Connection failed: " . mysqli_connect_error()); } echo "Connected successfully"; mysqli_close($conn); {% endhighlight %}

The main method used in this script is the mysqli_connect() method. The mysqli_connect() method opens a new connection to the MySQL server. The syntax is: mysqli_connect (host, username, password, dbname, port, socket);.
